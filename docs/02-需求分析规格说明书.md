## 2. 系统概述

### 2.1 项目背景

家教服务系统是一个面向家教和学生/家长的供需匹配平台。该系统旨在解决家教求职和学生找家教之间的信息不对称问题，提供一个高效的信息发布与匹配渠道。

### 2.2 系统目标

1. 帮助家教发布个人授课信息，扩大曝光度
2. 帮助学生/家长快速找到合适的家教
3. 提供智能匹配功能，提高匹配效率
4. 管理员后台管理用户账号

### 2.3 系统特性

- 用户注册与登录
- 信息发布与修改
- 双向搜索与筛选
- 管理员后台管理

## 4. 功能需求

### 4.1 用户注册与登录

#### 4.1.1 用户注册 (US-01)

**描述**：用户注册账号并设置密码

**前置条件**：用户未注册过账号

**操作流程**：
1. 用户进入首页，点击"我是家教"或"我是学生/家长"
2. 系统跳转到注册页面
3. 用户填写：昵称、手机号、邮箱、密码、确认密码
4. 系统校验信息有效性
5. 注册成功，跳转到登录页面

**业务规则**：
- 昵称必须唯一
- 密码长度≥8位
- 两次密码必须一致

**异常处理**：
- 昵称已存在：提示"昵称已存在，请更换"
- 密码不一致：提示"两次密码不一致"
- 密码少于8位：提示"密码至少8位"

#### 4.1.2 用户登录 (US-01)

**描述**：已注册用户登录系统

**前置条件**：用户已注册账号

**操作流程**：
1. 用户输入昵称和密码
2. 系统验证账号密码
3. 登录成功跳转主页

**异常处理**：
- 用户不存在：提示"用户不存在"
- 密码错误：提示"密码错误"
- 账号已注销：提示"账号已注销"

### 4.2 管理员登录 (US-02)

**描述**：管理员登录后台

**前置条件**：管理员知道隐藏入口URL

**操作流程**：
1. 管理员访问 /admin/login
2. 输入用户名和密码
3. 登录成功跳转后台主页

**账号信息**：
- 用户名：admin
- 密码：admin123

### 4.3 家教信息发布 (US-03)

**描述**：家教发布个人授课信息

**前置条件**：家教已登录

**操作流程**：
1. 进入个人信息页面
2. 填写/修改授课信息
3. 点击保存

**信息字段**：
| 字段 | 类型 | 说明 |
|------|------|------|
| 头像 | 图片 | jpg格式，≤2MB |
| 性别 | 单选 | 男/女 |
| 擅长科目 | 多选 | 至少1科 |
| 授课年级 | 多选 | 至少1年级 |
| 授课方式 | 多选 | 线上/线下 |
| 期望费用 | 数字 | 100-500元/节 |
| 授课时间 | 时间段 | 每周可多选 |
| 自我介绍 | 文本 | ≤50字 |
| 微信号 | 文本 | 非必填 |
| 接受异性 | 布尔 | 默认是 |

**异常处理**：
- 头像过大：提示"头像文件过大"
- 必填字段未填：提示对应错误
- 费用超范围：提示"费用范围为100-500元"

### 4.4 学生需求发布 (US-04)

**描述**：学生/家长发布家教需求

**前置条件**：学生已登录

**操作流程**：
1. 进入需求信息页面
2. 填写/修改需求信息
3. 点击保存

**信息字段**：
| 字段 | 类型 | 说明 |
|------|------|------|
| 头像 | 图片 | jpg格式，≤2MB |
| 性别 | 单选 | 男/女 |
| 科目 | 多选 | 至少1科 |
| 年级 | 单选 | 必选 |
| 授课方式 | 多选 | 至少1种 |
| 预算上限 | 数字 | 无限制 |
| 上课时间 | 时间段 | 每周可多选 |
| 补充要求 | 文本 | ≤50字 |
| 微信号 | 文本 | 非必填 |
| 接受异性 | 布尔 | 默认否 |

### 4.5 搜寻家教 (US-05, US-06)

**描述**：学生/家长浏览并筛选家教列表

**前置条件**：学生已登录，已填写需求信息

**操作流程**：
1. 进入"搜寻家教"页面
2. 系统自动根据科目、年级匹配
3. 设置筛选条件（排序、授课方式、预算范围）
4. 点击"应用筛选"
5. 浏览列表，点击查看详情

**筛选条件**：
- 排序方式：最新/最少薪酬
- 授课方式：线上/线下
- 预算范围：最低~最高

### 4.6 搜寻生源 (US-07)

**描述**：家教浏览学生需求列表

**前置条件**：家教已登录，已填写授课信息

**操作流程**：
1. 进入"搜寻生源"页面
2. 系统自动根据科目、年级匹配
3. 设置筛选条件
4. 点击"应用筛选"
5. 浏览列表，点击查看详情

### 4.7 查看详情 (US-06, US-07)

**描述**：查看对方完整信息

**操作流程**：
1. 在列表页点击"查看详情"
2. 显示完整信息页面

**显示信息**：
- 头像、昵称、性别
- 科目、年级、授课方式
- 费用/预算
- 授课时间
- 自我介绍/补充要求
- 微信号

### 4.8 用户注销 (US-09)

**描述**：用户注销账号

**前置条件**：用户已登录

**操作流程**：
1. 点击"注销账号"
2. 弹出确认对话框
3. 确认后账号失效
4. 跳转到首页

**业务规则**：
- 账号失效，数据保留
- 注销后无法恢复
- 再次登录提示"账号已注销"

### 4.9 管理员后台 (US-08)

**描述**：管理员管理用户账号

**前置条件**：管理员已登录

**操作流程**：
1. 查看用户列表
2. 可按昵称/注册时间/身份类型排序
3. 点击"修改密码"
4. 输入新密码≥8位
5. 密码更新成功

## 5. 非功能需求

| 项目 | 要求 |
|------|------|
| 响应时间 | 页面加载≤3秒 |

| 项目 | 要求 |

| 项目 | 要求 |
|------|------|
| 浏览器兼容 | Chrome、Firefox |
