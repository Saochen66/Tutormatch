# 边界类 / 控制类 / 实体类

## 边界类

| 中文类名 | 英文类名 | 类的作用概述 |
| --- | --- | --- |
| 首页界面 | IndexPage | 系统入口，提供注册与管理员入口导航 |
| 登录界面 | LoginPage | 用户登录与身份校验入口 |
| 注册界面 | RegisterPage | 家教/学生注册入口 |
| 家教信息界面 | TutorInfoPage | 家教信息展示与填写界面 |
| 学生需求界面 | StudentInfoPage | 学生需求展示与填写界面 |
| 搜寻家教界面 | SearchTutorsPage | 学生端筛选家教与收藏入口 |
| 搜寻生源界面 | SearchStudentsPage | 家教端筛选生源与收藏入口 |
| 家教详情界面 | TutorDetailPage | 展示家教详细信息 |
| 学生详情界面 | StudentDetailPage | 展示学生详细信息 |
| 我的中心界面 | MyCenterPage | 修改信息、撤回需求、收藏管理入口 |
| 我的收藏界面 | MyFavoritesPage | 展示与管理收藏对象 |
| 管理员登录界面 | AdminLoginPage | 管理员登录入口 |
| 管理员面板界面 | AdminDashboardPage | 管理员用户管理界面 |
| AI 助手界面 | AssistantPage | 智能助手对话与返回入口 |

## 控制类

| 中文类名 | 英文类名 | 类的作用概述 |
| --- | --- | --- |
| 认证控制器 | AuthController | 用户注册、登录、登出、加载用户身份 |
| 家教信息控制器 | TutorController | 家教信息增改与展示 |
| 学生需求控制器 | StudentController | 学生需求增改与展示 |
| 搜索控制器 | SearchController | 家教/生源双向筛选与排序 |
| 收藏控制器 | FavoriteController | 收藏切换与收藏列表展示 |
| 推荐匹配控制器 | MatchController | 计算最佳匹配及匹配度说明 |
| 撤回需求控制器 | ListingController | 撤回/重新发布需求切换 |
| 管理员控制器 | AdminController | 管理员登录与用户管理 |
| AI 助手控制器 | AssistantController | 组装上下文并调用大模型或本地兜底 |

## 实体类

| 中文类名 | 英文类名 | 类的作用概述 |
| --- | --- | --- |
| 用户 | User | 系统账号基础信息与角色 |
| 家教信息 | TutorInfo | 家教资料与授课偏好 |
| 学生需求 | StudentInfo | 学生/家长需求信息 |
| 管理员 | Admin | 管理员账号与权限 |
| 收藏 | Favorite | 用户收藏记录（对家教或生源） |
